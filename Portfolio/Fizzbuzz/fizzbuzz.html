Fizzbuzz:

Create a function that iterates through 100 numbers.
When a number is divisible by 3, print the word "Fizz".
When a number is divisible by 5, print the word "Buzz".
When a number is divisible by both 3 and 5, print the word "Fizzbuzz".

Code:

const fbCount = 100;
const fizz = 3;
const buzz = 5;
const output = document.getElementById("output");

for (let x = 1; x <= fbCount; x++) {

(x % fizz == 0 && x % buzz == 0) ? output.innerHTML += `<li>Fizzbuzz</li>`: null; //Divisible by fizz and buzz

(x % fizz == 0 && x % buzz != 0) ? output.innerHTML += `<li>Fizz</li>`: null; //Divisible by Fizz

(x % fizz != 0 && x % buzz == 0) ? output.innerHTML += `<li>Buzz</li>`: null; //Divisible by Buzz

(x % fizz != 0 && x % buzz != 0) ? output.innerHTML += `<li>${x}</li>`: null; //Divisible by netiher. (Print number)


}
